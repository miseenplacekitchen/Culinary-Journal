STRIPE SETUP — PLAIN ENGLISH (optional; skip until you want card payments)
══════════════════════════════════════════════════════════════════════
What it is: Lets members pay on the site with a card and get upgraded
automatically. Without it, nothing breaks — you keep manual upgrades.

What you do NOT need to do today: All four Stripe steps below.

When you ARE ready (you need a Stripe account at stripe.com first):

Step 1 — Deploy edge functions (upload two small server programs)
  Where: Supabase Dashboard → Edge Functions
  What: Deploy create-checkout-session and stripe-webhook
  (Agent can do this for you if you prefer not to use terminal.)

Step 2 — Set secrets (store Stripe keys safely in Supabase, not in files)
  Where: Supabase Dashboard → Edge Functions → Secrets
  Add: STRIPE_SECRET_KEY = your Stripe secret key (sk_live_... or sk_test_...)
  Add: STRIPE_WEBHOOK_SECRET = from Step 4 below

Step 3 — Turn on Stripe in your admin panel
  Where: theculinaryjournal.site → Dashboard → Finance → Pricing
  Check "Enable Stripe", fill in publishable key (pk_...) and prices, Save.

Step 4 — Tell Stripe where to send payment confirmations
  Where: stripe.com Dashboard → Developers → Webhooks → Add endpoint
  URL: https://kzywmodvfbyexqgipcjt.supabase.co/functions/v1/stripe-webhook
  Event: checkout.session.completed
  Copy the "Signing secret" → paste into Step 2 as STRIPE_WEBHOOK_SECRET
