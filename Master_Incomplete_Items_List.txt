How to read this document (legacy)

Every item below is something the roadmap or the issues list promises that the current repository does not deliver, or delivers only partially. Each item carries a status verified against the actual code on 6 June 2026:

[MISSING] — no implementation found anywhere in the repository (verified by code search).

[PARTIAL] — a page or hook exists, but the promised capability is absent or incomplete; the specific gap is stated.

[VERIFY] — code exists but live behaviour cannot be confirmed from a code audit alone; needs a workflow test in the live environment (most are blocked behind the admin panel fix).

This is a code-level audit of the repository, not a click-through of the live site. Per the agreed working rules, no item closes until it is owner-verified live and its workflow test passes — "page renders" is not "done".
